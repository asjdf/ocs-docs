油猴安装教程

一键安装​

打开你的浏览器，安装油猴浏览器拓展， 安装途径有如下：

  • 油猴官网

  • 谷歌浏览器网上商店(需翻墙)
  • Edge 浏览器网上商店

手动安装​

觉得麻烦的话可以下载OCS软件，软件中自带油猴拓展。

1.下载作者提供的第三方油猴文件

https://cdn.ocsjs.com/resources/Tampermonkey.zip

2.将整个文件解压缩

3.根据使用的浏览器查看对应教程

  • 谷歌浏览器安装
  • Edge浏览器安装

谷歌浏览器安装​

4.最后选择解压缩后的文件即可实现油猴安装。

Edge 浏览器安装​

4.最后选择解压缩后的文件即可实现油猴安装。
